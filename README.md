# PLATIFY Landing Page

Landing page profesional para PLATIFY - Sistema de Costeo Gastronómico con IA

## 🚀 Deploy

### Opción 1: Netlify Drop (Más Fácil)
1. Ve a https://app.netlify.com/drop
2. Arrastra toda la carpeta `platify-landing`
3. ¡Listo! Tendrás una URL tipo `https://platify-landing-xyz.netlify.app`

### Opción 2: Netlify CLI
```bash
npm install -g netlify-cli
cd platify-landing
netlify deploy
```

### Opción 3: GitHub + Netlify
1. Crea repo en GitHub
2. Push esta carpeta
3. Conecta con Netlify
4. Auto-deploy activado

## 🎨 Personalización

### Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --primary: #D4A93A;
    --primary-dark: #B8860B;
    --secondary: #1a1a2e;
}
```

### Contenido
- Pricing: Edita la sección `#pricing` en `index.html`
- Features: Modifica `.features-grid`
- Contact: Actualiza el email en el footer
